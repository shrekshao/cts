  • pp(strings: TemplateStringsArray, ...values: ReadonlyArray<Directive | string | number>): string
    • Defined in src/common/util/preprocessor.ts:110

    A simple template-based, non-line-based preprocessor implementing if/elif/else/endif.

    example
        const shader = pp`
${pp._if(expr)}
  const x: ${type} = ${value};
${pp._elif(expr)}
${pp.__if(expr)}
...
${pp.__else}
...
${pp.__endif}
${pp._endif}`;
